# Random Stoic Quotes
A Random Stoic Quotes generator that consumes [JSON API v1](http://jsonapi.org) compliant data from [Random Stoic Quotes API](https://randomstoicquotesapi.herokuapp.com).

## Features
* Click to random
* Click to tweet

## Built With
* Javascript (ECMAScript2015+)
